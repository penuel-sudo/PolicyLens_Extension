Example:

3. Design Rules (To Avoid AI-Generic Look)

This is where you kill:

generic gradients

basic layouts

flat boring UI

Example:

Avoid generic AI gradients, default Tailwind spacing, predictable section layouts, and common hero patterns.
Use unique layout rhythm, broken grids, breathing whitespace, asymmetric composition, and intentional imbalance.

4. UI + UX Flow Direction

This defines how users emotionally move through the page.

Example:

Design the UX flow as a story:
Confusion → Clarity → Excitement → Trust → Desire → Action.

or

Structure the flow to psychologically pull users downward using curiosity gaps, visual rhythm, and progressive emotional build-up.

5. Motion + Micro-Interactions Layer

This is what makes designs feel alive and premium.

Example:

Use subtle glassmorphism, micro hover effects, scroll-triggered animations, soft easing curves, floating elements, gradient noise textures, blur transitions, and gentle parallax depth.

6. Conversion Psychology Layer

This separates pretty sites from money sites.

Example:

Apply conversion psychology: clarity over cleverness, emotional triggers, visual hierarchy for scanning, trust stacking, social proof placement, and frictionless CTA positioning.
